BRICS does no longer actively develop the <bigwig> system. Instead, we are working on JWIG, which is a Java-based successor to <bigwig>.

  • Go to the JWIG site.

<bigwig> is a high-level programming language for developing interactive Web services.

The goal of the <bigwig> project is to remove many of the obstacles that face current developers of Web services to lower cost while increasing functionality and reliability, compared to CGI/Perl scripts, ASP, or PHP based solutions. The <bigwig> language is a descendant of the Mawl project, inheriting the concepts of sessions and document templates.

<bigwig> is being developed at the BRICS research center at the Department of Computer Science, Aarhus University, Denmark.

If this is your first visit to the <bigwig> home page we encourage you to read the introduction.

The latest release of the <bigwig> compiler and runtime system is version 2.0. It is available for download under the GNU Public License. It runs on Linux/Solaris systems with the Apache Web server.

If you are a DAIMI student, you have access to freewig.brics.dk for compiling and hosting your own <bigwig> services. If you are not a DAIMI student but wish to have access to freewig, send us an email.
